About

This system was created, with love, by Cameron O'Connor, JCR Internet Officer 2018-19. It is released under the MIT Licence, and the source is available on GitHub. The backend uses the Django framework, while the UI is simply built using Bootstrap and a little jQuery.

If you have any suggestions about how the system can be improved, please feel free to contact the Internet Officer at internet@jcr.dow.cam.ac.uk.
